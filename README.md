# Just A Jumper  
*by Cubes And Spikes*

Игра "Just A Jumper" — это уникальный платформер, вдохновлённый режимом платформера в Geometry Dash и использующий уникальные геймплейные фишки оттуда.  


---

## 1. Геймплей

- Персонаж может двигаться с помощью стрелок
- Нажатие вызывает либо прыжок, либо активацию полёта
- Отпускание кнопки завершает действие.

---

## 2. Персонаж

- Основной игровой персонаж — шар.
- Дизайн может меняться визуально (темы, цвета), но базовая форма остаётся шарообразной.
- Касание с препятствием = смерть

---

## 3. Уровни

- Минимум 4–5 уровней в финальной версии.
- Обязательно включает уровень разработчиков.
- 1 уровень - обучение, простое геометрическое пространство
- 2-4 уровни - уникальные сеттинги(Недорешено, возможно лиминальное пространство, античность, киберпанк, стимпанк и тп)
- Уровни делаются из тайлов, взаимодействие с препятствиями и частями осуществляется через physics engine и особые условия

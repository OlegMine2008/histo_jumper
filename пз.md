# Пояснительная записка  
## к проекту «Just A Jumper»

---

## 1. Введение

Проект **«Just A Jumper»** представляет собой 2D-игру в жанре простого платформера, разработанную на языке программирования Python с использованием графической библиотеки **Arcade**.

Цель проекта — создание простого, но функционального игрового приложения, демонстрирующего навыки:
- объектно-ориентированного программирования;
- работы с графическим интерфейсом;
- организации игрового цикла;
- использования встроенных движков со своими правками;
- создании уровней в Tiled;

Проект выполнен в рамках учебной программы **Яндекс Лицея**.

---

## 2. Назначение проекта

Приложение предназначено для:
- демонстрации принципов разработки игр на Python;
- закрепления знаний по архитектуре программ;
- изучения работы с событиями клавиатуры;
- созданием уникальных объектов, не связанных напрямую с физическим движком;
- демонстрации возможностей программы Tiled

Игра может использоваться как учебный пример или как простая игра для пользователей.

---

## 3. Используемые технологии

В проекте использованы следующие технологии и инструменты:

- **Python 3** — основной язык программирования.
- **Arcade** — библиотека для создания 2D-игр.
- **Time** - библиотека для управления временем.
- **Object-Oriented Programming (OOP)** — организация логики игры.

## 5. Описание файлов проекта

### 5.1 `main.py`

Файл запуска приложения.

Функции:
- создание игрового окна;
- установка размеров экрана;
- запуск главного цикла Arcade;
- отображение стартового меню.

---

### 5.2 `menu.py`

Отвечает за пользовательский интерфейс меню.

Реализует:
- стартовое меню игры;
- обработку нажатия мыши;
- переход к игровому экрану;

---

### 5.3 `platformer.py`

Основной файл игровой логики.

Содержит:
- игровой цикл (`on_update`, `on_draw`);
- управление персонажем;
- обработку столкновений и касаний;
- завершение игры и переход в меню.

---

## 6. Управление в игре

- **W** или **стрелка вверх** или **SPACE** — прыжок
- **A** или **стрелка влево** — движение влево  
- **D** или **стрелка вправо** — движение вправо  

---

## 7. Инструкция по запуску

1. Установить Python версии **3.10 и выше**.

2. Установить зависимости:
`pip install arcade`
`pip install time`

3. Запустить проект:
`python main.py`

4. Или установить main.py как exe файл:
`pip install pyinstaller` - установщик .py файлов в .exe
`pyinstaller main.py` - установит папку dist, в ней будет .exe файл и все зависимости
